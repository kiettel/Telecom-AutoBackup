File Menu:

Configure selected service For the TFTP Server, FTP Server and Syslog server, brings up the configuration dialog.
Start/Stop Selected service Starts or stops the currently selected service (whichever service is active in the current view).
Toggle Logging Starts or stops logging for the currently selected service (whichever service is active in the current view).
Toggle Debugging Starts or stops debugging for the currently selected service (whichever service is active in the current view).
Clear Display For the TFTP Server, FTP Server and Syslog server,  clears the display list of any past sessions. For the TFTP Client, resets the form to the default values.
Exit Exits the program

View Menu:

Status Bar Toggles the display of the status bar on or off.
Tftp Server Makes the TFTP Server the active view.
Ftp Server Makes the FTP server the active view
Syslog Server Makes the Syslog Server the active view
Tftp Client Makes the TFTP Client the active view

Operations Menu (TFTP Client only):

Start Transfer Starts the currently configured transfer
Stop Transfer Stops the transfer that is currently in progress. Optionally, allows you to cancel all pending transfers
Send Files Allows you to select a group of files which will be sent to the host sequentially
Send Directory Allows you to choose a directory. All of the files in the directory (plus subdirectories) will be sent to the host sequentially.

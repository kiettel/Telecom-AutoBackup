When you click this button, the configuration screen for the current service is displayed
When you click this button, the current service is started. In the case of the TFTP Client, the configured transfer is initiated
When you click this button, the current service is stopped. In the case of the TFTP Client, the configured transfer is halted.
If you see this icon, it means that the current service is being logged. Click the icon to disable logging
If you see this icon, it means that the current service is not being logged. Click the icon to enable logging
If you see this icon, it means that the current service is being debugged. Click the icon to disable debugging
If you see this icon, it means that the current service is not being debugged. Click the icon to enable debugging
Clicking this icon will clear all values for the current service. In the case of TFTP Server, FTP Server and Syslog Server, it will clear information in the list control. In the case of TFTP client, it will reset the form back to it's default values.
Clicking this button will allow you to view any log (*.log) or debug (*.dbg) files generated by 3CDaemon. The files will be loaded in your default Windows text editor.
Clicking this button will allow you to send a group of files via the TFTP Client. Make sure that all other relevant values have been completed on the TFTP Client form before selecting this option.
Clicking this button will allow you to send the all the files in a   directory via the TFTP Client. Make sure that all other relevant values have been completed on the TFTP Client form before selecting this option.
